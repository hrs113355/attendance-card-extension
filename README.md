# 補卡助手 Chrome Extension

這是一個 Chrome 擴充功能，用於自動化人資系統的補卡作業。

## 功能特色

- **設定頁面**：儲存員工 ID、補卡理由、上下班時間等設定
- **月曆介面**：直觀的日期選擇介面
- **自動月份判斷**：可設定根據當前日期自動判斷要補卡的月份
- **批量補卡**：一次選擇多個日期，自動送出上下班打卡
- **隨機時間**：避免每天都相同時間的問題

## 安裝方式

1. 打開 Chrome 瀏覽器
2. 前往 `chrome://extensions/`
3. 開啟右上角的「開發人員模式」
4. 點選「載入未封裝項目」
5. 選擇這個資料夾 (`attendance-card-extension`)

## 使用方式

### 首次設定
1. 安裝完成後，右鍵點選擴充功能圖示 → 選項
2. 設定以下資訊：
   - **員工 ID**：你在人資系統中的員工編號
   - **補卡理由**：例如「補卡」
   - **上下班時間**：設定固定的上下班小時
   - **自動月份**：勾選後會自動判斷月份（15號前=上月，16號後=下月）

### 日常使用
1. 登入目標人資系統網站
2. 點擊瀏覽器工具列上的補卡助手圖示
3. 在月曆中點選需要補卡的日期
4. 點擊「送出補卡申請」按鈕
5. 等待處理完成

## 注意事項

- 此擴充功能僅在特定人資系統網站上運作
- 請確保在人資系統頁面中使用，需要有效的 session
- 每個日期會自動補上班和下班兩次卡
- 分鐘數會隨機生成，避免時間過於規律
- 建議一次不要選擇過多日期，避免系統負荷過大

## 檔案結構

```
attendance-card-extension/
├── manifest.json          # 擴充功能設定檔
├── popup.html             # 主要介面 HTML
├── popup.js               # 主要介面邏輯
├── options.html           # 設定頁面 HTML
├── options.js             # 設定頁面邏輯
├── content.js             # 內容腳本，處理實際請求
├── icons/                 # 圖示資料夾
└── README.md              # 說明文件
```

## 技術細節

- 使用 Manifest V3
- 透過 Content Script 注入頁面執行 XHR 請求
- 使用 Chrome Storage API 儲存設定
- 支援自動月份判斷和手動月份選擇
